💾 Data Management

Your data is auto-saved to Google Drive whenever you make changes. Use these tools to download a local backup, restore from a previous backup file, or reset the dashboard to a clean state.

⬇ Download Backup
Saves all your data as a .json file to your Downloads folder. Filename includes today's date.
⬆ Load from File
Replaces all current data with the contents of a backup .json file. This cannot be undone.
⚠ Clear All Data
Resets the dashboard to a completely empty state. All records will be permanently deleted. Download a backup first.
