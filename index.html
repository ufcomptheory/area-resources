💾 Data Management

Your data is auto-saved to Google Drive whenever you make changes. Use these tools to download a local backup, restore from a previous backup file, or reset the dashboard to a clean state.

⬇ Download Backup
Saves all your data as a .json file to your Downloads folder. Filename includes today's date.
⬆ Load from File
Replaces all current data with the contents of a backup .json file. This cannot be undone.
⚠ Clear All Data
Resets the dashboard to a completely empty state. All records will be permanently deleted. Download a backup first.

🗓 Archive Settings

Set the calendar to archive from. This should be the Comp Studio calendar (ufcomposers@gmail.com). Add it to your Named Calendars in Sign-Up Sheets → Settings, then select it here.

Set as ARCHIVE_CALENDAR_ID in Script Properties for the scheduled auto-archive.
⏱ Automatic Archive Schedule

Set up two time-based triggers in Apps Script to auto-archive each semester:

FunctionTrigger TypeDayWhat It Archives
archiveFallSemesterMonth timer1st of month (runs Jan 1, skips other months automatically)Aug 1 – Dec 31 of previous year → "Fall YYYY"
archiveSpringAndSummerMonth timer1st of month (runs Aug 1, skips other months automatically)Jan 1 – Jul 31 of current year → "Spring YYYY"

In Apps Script: Triggers (clock icon) → + Add Trigger. Set source to Time-driven, type to Month timer, day of month 1, time 1am–2am. The functions check internally whether it's the right month and exit immediately if not.

Also set ARCHIVE_CALENDAR_ID in Script Properties to the calendar ID.
